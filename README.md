# Image Renamer Pro Web (LCA Picture Rename)

LCA（ライフサイクルアセスメント）データ収集のための画像リネームWebアプリケーションです。
デスクトップアプリ版の機能をWebブラウザ上で実現し、Excelマスタ連携やZIP一括ダウンロードに対応しています。

## 主な機能

- **マスタデータ読込**: 素材マスタおよび加工マスタのExcelファイル（.xlsx）を読み込み可能
- **画像プレビュー**: 選択した画像を表示しながら情報入力が可能
- **詳細入力**: 部品名、重量、素材、加工方法などの属性情報を入力
- **自動リネーム**: 入力された属性情報に基づき、規定のフォーマットでファイル名を変更
- **一括ダウンロード**: リネーム後の全画像をZIPファイルとしてダウンロード

## 使用方法

1. `index.html` をWebブラウザで開きます。
2. **素材マスタ読込**ボタンから素材一覧のExcelファイルを読み込みます。
3. **加工マスタ読込**ボタンから加工一覧のExcelファイルを読み込みます。
4. **画像を選択**ボタンからリネームしたい画像ファイル（複数可）を選択します。
5. 左側のパネルで各画像に対する情報を入力します。
   - 部品名、重量、単位、素材区分、素材名、加工方法などを選択・入力
6. **適用 & 次へ**（またはEnterキー）で入力を確定し、次の画像へ進みます。
7. 全ての入力が完了したら、**Rename & Download ZIP**ボタンをクリックして保存します。

## 動作環境

- モダンブラウザ（Chrome, Edge, Safari, Firefoxなど）
- JavaScript有効化必須
