# 📸 Image Renamer Pro - Web版

## 概要

Image Renamer Pro のWebブラウザ版です。HTML、CSS、JavaScriptのみで動作する完全サーバーレスのアプリケーションです。

## 特徴

✨ **サーバーレス**: 完全にブラウザ内で動作、サーバー不要  
📊 **Excelサポート**: IDマスター（.xlsx）から素材、加工方法、実施者を読み込み  
🖼️ **複数画像対応**: 複数の画像ファイルを一括処理（JPEG、PNG、GIF、WebP、HEIC対応）  
💾 **ZIP出力**: リネーム済みファイルをZIPでダウンロード  
🎨 **モダンUI**: レスポンシブ対応の美しいインターフェース  
🚀 **高速処理**: すべての処理がブラウザ内で完結  
📱 **HEIC対応**: iPhoneで撮影したHEIC画像もそのまま処理可能（Safariでプレビュー対応）

## 使い方

### 1. ファイルを開く

`web/index.html` をダブルクリックしてブラウザで開きます。

### 2. IDマスターを読み込み

「IDマスター.xlsx」ファイルを選択します。以下の3つのシートが必要です：

#### 素材シート
| 素材名 | 素材ID | 素材区分 |
|--------|--------|----------|
| 鉄 | M001 | 金属 |
| アルミ | M002 | 金属 |

#### 加工シート
| 加工方法名 | 加工ID |
|------------|--------|
| 切削 | P001 |
| 溶接 | P002 |

#### 実施者シート
| 実施者名 | 実施者ID |
|----------|----------|
| 山田太郎 | U001 |
| 佐藤花子 | U002 |

### 3. 画像ファイルを選択

複数の画像ファイルを選択します（JPG、PNG、HEICなど）。

### 4. 各画像の情報を入力

- 番号（自動設定）
- 実施者
- 部品名
- 重量
- 単位
- 素材区分
- 素材名
- 加工方法
- 写真区分
- 特記事項

「適用 & 次へ」ボタンで次の画像に進みます。

### 5. ZIPファイルをダウンロード

すべての画像を処理したら、「リネーム済みファイルをダウンロード」ボタンでZIPファイルをダウンロードします。

## ファイル名形式

```
番号_部品名_重量_単位_素材ID_加工ID_実施者ID_写真区分_特記事項.拡張子
```

例：
```
1_ボルト_0.5_kg_M001_P001_U001_P_0.jpg
1_ナット_0.3_kg_M002_P002_U001_M_1.jpg
2_ワッシャー_0.1_kg_M001_P001_U002_P_0.png
```

## 技術仕様

### 使用ライブラリ

- **SheetJS (xlsx.js)**: Excelファイル読み込み
- **JSZip**: ZIP生成
- **FileSaver.js**: ファイルダウンロード
- **heic2any**: HEIC→JPEG変換（プレビュー用）
- **libheif-js**: HEICデコーダ（フォールバック用）

### 対応画像形式

| 形式 | 拡張子 | プレビュー | リネーム・ZIP |
|------|--------|-----------|--------------|
| JPEG | .jpg, .jpeg | ✅ 全ブラウザ | ✅ |
| PNG | .png | ✅ 全ブラウザ | ✅ |
| GIF | .gif | ✅ 全ブラウザ | ✅ |
| WebP | .webp | ✅ 全ブラウザ | ✅ |
| BMP | .bmp | ✅ 全ブラウザ | ✅ |
| HEIC/HEIF | .heic, .heif | ⚠️ Safariのみ | ✅ |

### ブラウザ要件

| ブラウザ | バージョン | HEICプレビュー | 備考 |
|---------|-----------|---------------|------|
| Safari | 14+ | ✅ 対応 | **⭐ HEIC利用時は推奨** |
| Chrome | 90+ | ❌ 非対応 | HEVCライセンスの制限 |
| Firefox | 88+ | ❌ 非対応 | |
| Edge | 90+ | ❌ 非対応 | |

> **注意**: Chrome/Firefox/EdgeでもHEICファイルのリネーム・ZIP保存は正常に動作します。プレビューのみ表示されません。

### セキュリティ

すべての処理がブラウザ内で完結し、データは外部に送信されません。

## トラブルシューティング

### Excelファイルが読み込めない

- ファイル形式が .xlsx または .xls であることを確認
- 必要なシート（素材、加工、実施者）が存在することを確認
- 各シートに必要な列が存在することを確認

### 画像が表示されない

- 対応する画像形式（JPG、PNG、GIF、WebP、HEICなど）を使用
- ファイルサイズが大きすぎないか確認（1枚あたり10MB以下推奨）

### HEICファイルのプレビューについて

- **Safari**: ネイティブ対応のため正常にプレビュー表示されます ✅
- **Chrome/Firefox/Edge**: HEVCコーデックのライセンス制限により、プレビューは表示されません ❌
  - ただし、リネーム処理・ZIP保存は正常に動作します
  - プレビューが必要な場合はSafariをご利用ください
- 元ファイルは変換せず、そのままリネーム・ZIP化されます

### ZIPダウンロードができない

- ブラウザのポップアップブロックを確認
- ブラウザのダウンロード設定を確認
